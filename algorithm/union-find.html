素集合データ構造(Union-Find)

説明

集合を高速に扱うためのデータ構造。集合を合併する操作(unite), ある要素がどの集合に属しているか(find)を問い合わせる操作を行うことが出来る。

使い方

  • unite(x, y): 集合 $x$ と $y$ を併合する. 併合済のとき $false$, 未併合のとき $true$ が返される
  • find(k): 要素 $k$ が属する集合を求める
  • size(k): 要素 $k$ が属する集合の要素の数を求める

計算量

$O(\alpha(n))$

実装例

struct UnionFind
{
  vector< int > data;

  UnionFind(int sz)
  {
    data.assign(sz, -1);
  }

  bool unite(int x, int y)
  {
    x = find(x), y = find(y);
    if(x == y) return(false);
    if(data[x] > data[y]) swap(x, y);
    data[x] += data[y];
    data[y] = x;
    return(true);
  }

  int find(int k)
  {
    if(data[k] < 0) return(k);
    return(data[k] = find(data[k]));
  }

  int size(int k)
  {
    return(-data[find(k)]);
  }
};

応用 1: 2部グラフの頂点彩色

Union-Find を用いると $2$ 部グラフ判定が可能。頂点を倍長して偶奇に分ける。副作用として彩色もできる。

struct Bipartite_Graph : UnionFind
{
  vector< int > color;

  Bipartite_Graph(int v) : color(v, -1), UnionFind(v + v) {}

  void add_edge(int u, int v)
  {
    unite(u, v + color.size());
    unite(u + color.size(), v);
  }

  bool bipartite_graph_coloring()
  {
    for(int i = 0; i < color.size(); i++) {
      int a = find(i);
      int b = find(i + color.size());
      if(a == b) return (false);
      int c = color[a];
      if(c < 0) color[a] = 0, color[b] = 1, c = 0;
      color[i] = c;
    }
    return (true);
  }

  bool operator[](int k) const
  {
    return (bool(color[k]));
  }
};

応用 2: データ構造をマージする一般的なテク

データ構造をマージする一般的なテク(Weighted-Union-Heuristic) は, Union-Find の unite 操作で常に大きい木の根が全体の根になるよう連結する(union-by-rank) の考え方と同様。
